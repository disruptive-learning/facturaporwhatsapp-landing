# Factura por WhatsApp

Landing page para servicio de facturación por WhatsApp con asistente de IA.

## Descripción

Sitio web de marketing para un servicio de emisión de CFDI 4.0 mediante conversaciones de WhatsApp con un agente de inteligencia artificial.

## Tecnologías

- TypeScript
- Vite
- Tailwind CSS 4.x

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Estructura

- `index.html` - Página principal con hero, pricing, features y CTA
- `privacidad.html` - Política de privacidad
- `terminos.html` - Términos y condiciones
- `src/main.ts` - Lógica de modal de contacto y validación de teléfono
- `src/style.css` - Estilos con Tailwind CSS

## Características

- Diseño responsive
- Modal de captura de leads con validación de teléfono E.164
- Formulario con validación de términos
- Páginas legales con boilerplate para servicios de IA conversacional
- Animaciones CSS para chat mockup
